# Plano de execucao fase 1

## Objetivo

Montar o nucleo do agente imobiliario com alta organizacao, usando o que ja esta liberado no Codex.

## Fase 1 - Fundacao

1. Definir a estrutura oficial no Notion
2. Criar database de leiloes
3. Criar database de oportunidades imobiliarias
4. Padronizar campos, status e scores
5. Organizar entregas em markdown e CSV

## Fase 2 - Inteligencia

1. Consolidar score de oportunidade
2. Definir regra de CMA por m2
3. Criar logica de decisao preliminar
4. Criar formato executivo de relatorio

## Fase 3 - Operacao

1. Criar pipeline de ingestao por fonte
2. Deduplicar
3. Gravar no Notion
4. Gerar resumo
5. Preparar alertas

## Fase 4 - Escala

1. Rodar fora do Codex
2. Integrar APIs externas
3. Agendar execucoes automaticas
4. Ativar alertas por WhatsApp e email em producao

## Melhor caminho agora

Comecar pela base do Notion, porque ela vira o centro operacional do agente.
